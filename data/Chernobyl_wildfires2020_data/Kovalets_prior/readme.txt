gt.mat contains separate wildfires source, storm source and their sum (total).

Each source tensor has shape latitudes x longitudes x time (30).

Position of the first pixel is lon[1] = 27.75deg, lat[1] = 48.75deg, day = 1.

Created from tables S2 and S3 form Kovalets_2022_supl.docx.

data_gt.csv contains data from table S2. (wildfires)

storm_gt.csv contains data from table S3. (storm)
